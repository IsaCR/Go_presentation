** _Go is an open source programming language that makes it easy to build simple, reliable, and efficient software_ ** ![alt text](img/golang-logo.png)
## Historia Desarrollado en Google en el 2007 Sus creadores fueron: * Robert Griesemer * Rob Pike * Ken Thompson
## características * Go usa una sintaxis parecida a C. * Facilidad de lenguajes dinámicos como Python * Provee de un recolector de basura, reflexión y otras capacidades de alto nivel. * Admite el paradigma de programación orientada a objetos, pero no dispone de herencia de tipos. * Go no tiene excepciones.
## Versiones ![alt text](img/version1_6.png)
# ventajas * Sintaxis clara y estandarizada * El compilador forza a que se sigan buenas practicas de programación * El compilador es muy rápido * Facil de subir a producción * Soperte por concurrencia
# desventajas * Evidentemente es un lenguaje de un nivel de abstracción bajo/medio, que permite estar en contacto con los tipos de datos a nivel de procesador y usar los recursos de una forma eficiente * Si nos salimos de el tipo de aplicaciones para los que fue inventado, tendremos falta de bibliotecas potentes * La comunidad aún es muy pequeña

![alt text](img/RevelWhiteLines.png) Go-Revel
## Go-Revel, Caracteristias * Ultima versión: v0.12.0 * Requiere una version de Go: v1.3+ * Fácil de usar * Es un framework casi full stack, similar a Rails y Play * Patrón Modelo-Vista-Controlador
## Setup process 1. Descargar el archivo adecuado para instalar 2. Extraer el archivo en /usr/local 3. Crear un directorio donde se guardara el ambiente de desarrollo para Go 3. Define the GOPATH - export GOPATH=$HOME/Go 4. go get github.com/revel/cmd/revel - revel new app - revel run app
